# 无损压缩能否突破信息熵的限制

无损压缩就是数据在压缩后的还能原样恢复到压缩前的样子。

信息熵则被定义来衡量一段信息的信息量。

信息熵是熵编码的上限，但并非所有无损压缩算法的上限。[1]
